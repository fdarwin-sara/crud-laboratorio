# CRUD Laboratorio - Microscopios

Proyecto Full Stack para la gestión de inventario de microscopios.

---

## Tecnologías

- Frontend: Angular
- Backend: Spring Boot
- Base de datos: PostgreSQL

---

## Funcionalidades

- Crear microscopios
- Listar registros
- Editar registros
- Eliminar registros
- Validaciones en backend

---

## Arquitectura

- API REST con Spring Boot
- Frontend Angular consumiendo API
- Base de datos PostgreSQL

---

## Cómo ejecutar

### Backend
```bash
cd Backend_API_Microscopios
./mvnw spring-boot:run

### Frontend
cd Fronted_API
